# Galeri Gitar
Proyek Galeri Gitar ini adalah website responsif yang menampilkan koleksi gitar favorit. Pada halaman utama, terdapat video background pada area hero, galeri gitar (dengan 5 kartu gitar yang dapat diklik) untuk mengarahkan ke halaman detail masing-masing, serta section video, form kontak, dan footer. Setiap halaman detail menampilkan informasi tentang gitar tertentu dan menyediakan tombol untuk kembali ke beranda.

## Fitur
1. Responsive Design: Tampilan yang responsif menggunakan Flexbox dan media query.
2. Video Background: Area hero menggunakan video background yang diputar secara otomatis (autoplay, muted, loop).
3. Smooth Scrolling: Tombol "Lihat Koleksi" dan navigasi menggunakan efek smooth scrolling.
4. Halaman Detail: Setiap kartu gitar di galeri dapat diklik untuk melihat detail lebih lanjut pada halaman terpisah.
5. Form Kontak: Formulir kontak sederhana dengan validasi dan pesan sukses.
6. Konsistensi Header & Footer: Header dengan logo dan menu navigasi yang sama pada setiap halaman, serta footer dengan tautan sosial media.

## Cara Menjalankan
1. Clone atau Download repositori ini ke komputer Anda.
2. Pastikan struktur folder sesuai dengan yang tertera di atas agar path file (gambar, video, dan CSS) berfungsi dengan baik.
3. Buka file `index.html` di browser favorit Anda untuk melihat halaman utama.
4. Klik salah satu kartu gitar pada galeri untuk membuka halaman detail yang sesuai.
5. Pada halaman detail, gunakan tombol "Kembali ke Beranda"untuk kembali ke halaman utama.

## Teknologi yang Digunakan
1. HTML5 (index.html) untuk struktur halaman.
2. CSS (style.css) termasuk Flexbox dan media query untuk styling UI/UX dan responsivitas.
3. JavaScript (script.js) untuk smooth scrolling dan interaksi form kontak.
